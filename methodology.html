Assessment Framework

The Neurocognitive Assessment Suite follows a structured dimensional model of psychological screening. Each module integrates symptom frequency scaling, functional impairment indexing, and domain-threshold logic aligned with contemporary diagnostic frameworks.

1. Dimensional Measurement Model

All modules operate on a 0–4 frequency-based response format:

  • 0 – Never
  • 1 – Rarely
  • 2 – Sometimes
  • 3 – Often
  • 4 – Very Often

Raw scores are aggregated into domain-specific subscales. Dimensional percentages are calculated relative to theoretical maximums, allowing severity stratification independent of categorical diagnosis.

2. Subscale Architecture

Each screening module is divided into structured domains representing core symptom clusters. For example:

  • Executive Function & Attention: Inattention, Hyperactivity-Impulsivity, Functional Impairment
  • Mood Spectrum Index: Depressive Load, Irritability, Activation, Functional Impact

Subscale differentiation allows domain-specific interpretation rather than relying solely on total scores.

3. Domain Threshold Logic

In modules aligned with DSM-based constructs, threshold flags are applied when:

  • ≥ 4 symptom items within a domain are rated “Often” or “Very Often”

This threshold modeling approximates structured diagnostic domain criteria while preserving dimensional interpretation.

4. Severity Band Classification

Dimensional scores are categorized into severity bands:

  • 0–39% → Low
  • 40–69% → Moderate
  • 70%+ → Elevated

These bands provide structured interpretation while maintaining continuous score transparency.

5. Functional Impairment Index

Symptom frequency alone does not determine clinical significance. Each module includes impairment-specific items assessing:

  • Academic / occupational disruption
  • Interpersonal impact
  • Task completion difficulty
  • Daily functioning compromise

Impairment indices contextualize symptom burden within real-world performance domains.

6. Interpretive Model

Outputs are structured to provide:

  • Subscale dimensional percentages
  • Severity band classification
  • Domain threshold indicators
  • Activation–Depression polarity mapping (Mood module)

Interpretation is descriptive rather than diagnostic. Comprehensive clinical evaluation is recommended for high-severity profiles.
